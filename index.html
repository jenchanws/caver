CAVER Machine Shorthand System

This system was designed to be compatible with Stenotype (see Note 1) and Sokutaipu.

It can be used as a high-speed word processor, and it can also be used as a one-handed word processor, allowing you to enter text using touch typing with either your left or right hand.

(Note 1: the Stenotype layout is semi-symmetrical and will need to be modified by attaching some key tops.)
